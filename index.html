Tools and Projects

Velvyy Library

Velvyy adalah library pemrograman buatan sendiri yang menjadi fondasi berbagai proyekmu. Library ini memiliki fungsi utama seperti enkripsi (enc), dekripsi (desenc), desain teks dengan efek khusus (textCardDesign), dan koneksi multi-platform (WhatsApp, Telegram, Discord).

Fitur Utama:

  • Enkripsi dan Dekripsi data dengan algoritma custom
  • Desain teks aesthetic dengan warna dan tipografi unik
  • Modul koneksi multi-platform untuk integrasi bot dan aplikasi komunikasi

Manfaat:

Memudahkan pengembangan proyek berbasis bot dan aplikasi komunikasi dengan keamanan data yang terjamin dan tampilan UI yang menarik.

Bot WhatsApp Tsundere-Core

Bot WhatsApp dengan karakter tsundere yang imut, galak, dan centil. Bot ini dibangun menggunakan Node.js dan whatsapp-web.js dengan struktur modular plugin, memungkinkan penambahan fitur baru dengan mudah.

Fitur Utama:

  • Command interaktif dengan gaya bahasa tsundere khas
  • Auto-reply dan moderasi chat
  • Support fitur game mini, reminder, dan utilities

Manfaat:

Memberikan pengalaman chat yang menyenangkan dan interaktif, cocok untuk komunitas atau grup WhatsApp dengan tema anime dan kawaii.

VelvyXF Project

VelvyXF adalah proyek besar yang berisi kumpulan tools, bahasa pemrograman Velvyy, library, API key generator, dan berbagai source code. Proyek ini dirancang multi-platform dan modular untuk mendukung pengembangan aplikasi kompleks.

Fitur Utama:

  • Bahasa pemrograman Velvyy dengan sintaks gabungan Python dan JavaScript
  • Sistem modul dan import dengan standar khusus (@NerooXF/Velvyy+)
  • API key generator untuk pengamanan akses aplikasi

Manfaat:

Menyediakan platform pengembangan yang powerful dan fleksibel untuk pembuatan aplikasi custom dengan skalabilitas tinggi.

Bot Stalk Sosial Media

Bot yang bisa melakukan pencarian dan pengambilan data dari berbagai platform sosial media secara otomatis. Bot ini menampilkan hasil lengkap dengan keterangan jika data tidak ditemukan di suatu platform tanpa perlu input manual platform tertentu.

Fitur Utama:

  • Multi-platform (Instagram, TikTok, Twitter, YouTube, dll)
  • Otomatis menampilkan status "Not Found" jika data kosong
  • User-friendly dan efisien tanpa perlu memilih platform

Manfaat:

Memudahkan monitoring dan analisis data sosial media dalam satu alat tanpa harus cek satu per satu.
